# Fun.TS

TypeScriptで静的型付け言語の強みを活かした関数群。  
  
## 関数の説明

```js
// 引数2つが同値ないしは同じ参照先である場合に「true」を返します。
eq
// 「eq」関数の逆です。
ne
// 第一引数が第二引数よりも大きい場合に「true」を返します。
gt
// 第一引数が第二引数以上の場合に「true」を返します。
gtEq
// 第一引数が第二引数よりも小さい場合に「true」を返します。
lt
// 第一引数が第二引数以下の場合に「true」を返します。
ltEq

// 第一引数と第二引数の和を返します。
add
// 第一引数から第二引数を引いた値を返します。
reduce
// 第一引数と第二引数の積を返します。
times
// 第一引数を第二引数で割った値を返します。
divide
// 第一引数の第二引数乗を返します。
power

// 配列の先頭の要素を取得します。
fst
// 配列の二番目の要素を取得します。
snd
// 配列の最後の要素を取得します。
last

// 配列の中で最小の要素を返します。
min
// 配列の中で最大の要素を返します。
max

// 引数として受け取った真偽値の論理積を返します。
and
// 引数として受け取った真偽値の論理和を返します。
or
// 引数として受け取った真偽値の排他的論理和を返します。
xor
// 引数として受け取った真偽値を逆転させます。
not
// 引数として受け取った値が「true」っぽいかを返します。
truthy
// 引数として受け取った値が「false」っぽいかを返します。
falsy

// 関数を引数として受取り、引数に適用する関数を返します。
apply
// 自分自身を引数にとって適用する関数を返します。
rec

// 第二引数に指定した配列のうち、第一引数として受け取った関数の評価結果が「true」となる要素だけを抽出します。
// 第一引数の関数の引数には配列の各要素が渡されます。
filter
// 第二引数に指定した配列の各要素に、第一引数に渡した関数を実行した結果を取得します。
map
// 「map」関数の第一引数と第二引数の順番を入れ替えたものです。
looper
// 第一引数に渡した配列の要素に対して第二引数として渡した関数を適用した結果、「true」となる要素がひとつでもあれば「true」を返します。
any
// 第一引数に渡した配列の要素に対して第二引数として渡した関数を適用した結果、全ての要素の評価結果が「true」となれば「true」を返します。
all
// 第一引数で渡した配列のうち、第二引数の関数の評価結果が真となる要素の数を返します。
// 第二引数の関数は第一引数に配列の要素を、第二引数にインデックス番号を受け取ります。
countSatisfy
// 配列の中で最小の要素を取得します。
minFx
// 配列の中で最大の要素を取得します。
maxFx
// 第一引数で指定した配列の各要素を第二引数で指定した関数で評価結果を、第三引数で集計した結果を返します。
// デフォルトでは第三引数には合計を求めるadd関数が設定されています。
reducer

// 第一引数で指定したIDの配列から取得した要素を配列で返します。
getElms
// 第一引数で指定したタグ名の配列から生成したHTMLオブジェクトを配列で返します。
mkElms
// 第一引数で指定したタグ名の配列から生成したHTMLオブジェクト(SVG)を配列で返します。
mkElmSVGs
// 第一引数で指定した値を第二引数で指定した桁でゼロ埋めします。
zFill
// 引数として受け取ったオブジェクトをクエリパラメタ用にエンコードします。
URLencodeAssoc
// 第一引数以上、第二引数以下かを判定する関数を返します。
// 高階関数です。
between
// 第一引数の評価結果が真の場合には第二引数で渡された関数を、偽の場合には第三引数で渡された関数を実行します。
// 第四引数では実行される関数に渡される引数を指定できます。
switcher

// 受け取った値が正規表現を満たすか判定する関数を返します。
// 高階関数です。
regex
// 引数として受け取った値がメールアドレスの正規表現を満たすか判定します。
mailCheck
// 正規表現のサブマッチキャプチャを返します。
regexGrouping

// 指定したHTML要素の子要素をすべて削除します。
removeChildren

// 第一引数で指定したHTML要素の配列を順に第二引数で指定したHTML要素に挿入します。
append
// 改行タグを生成します。
mkBr
// 第一引数で指定した文字列の配列をテキストノードに変換し、順に第二引数で指定したHTML要素に挿入します。
// 各テキストノードの間には改行タグが挿入されます。
appendText
// 第二引数で指定した配列に第一引数で指定した配列の要素を順にプッシュしていきます。
push

// 第一引数で指定した回数分、第二引数で指定した関数を実行します。
// 第二引数の関数の引数にはインデックス番号が与えられます。
doNtimes

// 第一引数以上、第二引数以下の乱数を返します。
random

// 第一引数x第二引数の二次元配列を生成し、第三引数で指定した値で埋めます。
NxN
// 第一引数x第二引数の二次元配列を生成し、第三引数で指定した関数の戻り値で埋めます。
NxNfx

// 第一引数から第二引数までの範囲の配列を第三引数間隔(デフォルトは1)で生成します。
// 第四引数では同値(=)を含むかどうかを指定します。
fromAtoB

// 二次元配列を一次元配列に展開します。
flatter
// 第一引数で指定した配列と第二引数で指定した配列を交互に組み合わせた配列を生成します。
mixupMesh

// 引数で指定したクラスを持つHTML要素をすべて削除します。
removeClassifiedItems
```

## 環境構築手順

```shell
npm install -y yarn

mkdir ★★★
cd ★★★

yarn init -y

npm install --save-dev typescript @types/node
npx tsc --init

npm install eslint --save-dev
npx eslint --init

# コンパイル
npx tsc

# 常時コンパイル
tsc --watch
```
